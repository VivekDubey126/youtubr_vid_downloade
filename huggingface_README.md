# 📹 Youtubr - Full-Stack YouTube Downloader (HF Space)

This is a high-performance Next.js YouTube downloader running in a Docker container.

### Deployment on Hugging Face:
1. Create a [New Space](https://huggingface.co/new-space).
2. Set the name to `youtube-downloader`.
3. Choose **Docker** as the SDK.
4. Hugging Face will automatically build and run your app using the existing `Dockerfile`.
